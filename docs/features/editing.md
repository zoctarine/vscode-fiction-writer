
#### 1. Paragraphs

- Create new paragraph (empty lines) without hitting <kbd>enter</kbd> twice. This comes especially handy when writing short dialogue sentences: <kbd>enter</kbd> adds two empty lines, and <kbd>shift</kbd>+<kbd>enter</kbd> adds simple line break.
- in this case use ++shift+enter++ to create new paragraph, and <kbd>enter</kbd> to add normal line break.
- Distinguih between new lines and wrapped lines: Use *hanging-indent* for wrapped lines


#### 2. Folding

  - Get a outline view of all text: Fold paragraph sentences, leving only the first line visible.

#### 3. Dialogues

##### a) Markers

- Easily insert the dialogue marker (—): Automatically convert  `--` to prefered marker, either on compile time, or as you type it.

- Automatically add new dialogue marker, when creating a new paragraph from within a dialogue.

##### b) Indents

- Get a better view of dialogue paragraphs: automatically indent dialogue paragraph sentences.
