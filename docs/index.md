!!! danger "Important!"
    This extension is in early development(concept) phase. Use only for testing purposes.

## Main Features

- [Enhanced text editing](edit.md):
  - automatically insert new paragraph on ++enter++, or ++shift+enter++
  - solutions for writing dialogue using dialogue markers (like in some non-english languages), using either `em-dash` (—) or other marks.
  - autoreplace `-- ` with **—**
  - status bar quick action
  - ...

- [Format document](format.md)
  - clean up extra spaces
  - clean up extra lines
  - fix dialogue markers, indents
  - fix paragraph spacing
  - fix line spacing
  - ...

- [Export/Combine files](export.md)
  - combine multiple `.md` files into one single file.
  - export `.md` to other formats (like `.doc` or `.docx`)
    - uses [pandoc](https://pandoc.org/installing.html)
  - ...

- [Analyze text](stats.md)
  - count words, phrases, word repetitions
  - view document statistics
  - ...

- [View](view.md)
  - fold paragraph lines
  - tag files (view badges in file explorer)
  - writing mode (different theme, different font size)
  - text highlighting
  - typewriter mode
  - ...

- [Metadata](metadata.md)
  - use markdown `yaml` metadata blocks to categorize documents
  - parse metadata into a tree view
  - add badges and colors in **Explorer**
